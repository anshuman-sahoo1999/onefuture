# Netlify CMS - Quick Reference

## Access CMS
`https://yoursite.netlify.app/admin`

## Collections

### Advisors
- Name, Title, Photo, Biography
- Display Order (1, 2, 3...)
- Active toggle

### Gallery
- Title, Image, Caption
- Category, Date, Featured

### Downloads
- Title, Description, File
- Category, Date, File Size

### Site Settings
- Contact info
- Social media links
- Footer text

## Workflow
1. Create/Edit content
2. Save as Draft (optional)
3. Publish
4. Wait 1-2 min for deploy

## Media Folder
Uploaded files go to: `/images/cms/`

## Local Testing
```bash
netlify dev
```
Then visit: `http://localhost:8888/admin`
